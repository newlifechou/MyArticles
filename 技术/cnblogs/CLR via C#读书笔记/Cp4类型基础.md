# Cp4类型基础
## Object类型
Object是所有类型的基类，有Equals,GetHashCode,ToString,GetType四个公共方法，其中GetHashCode,ToString可以override. 受保护方法有MemeberwiseClone,Finalize.  
所有对象必须用new来创建，引用可以看成是安全的指针，指向真实对象在内存中的地址。
